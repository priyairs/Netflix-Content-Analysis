# Netflix Content Analysis

## Tools Used
- MySQL — Data cleaning & SQL analysis (15 queries)
- Python (Pandas, Seaborn, Matplotlib) — EDA & visualizations
- Excel — Interactive dashboard with pivot charts
- Tableau Public — Interactive global content map

## Key Insights
- 70% Movies vs 30% TV Shows
- US and India dominate Netflix content
- Content additions peaked in 2019
- TV-MA is the most common rating
